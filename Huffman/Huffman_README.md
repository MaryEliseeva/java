# Huffman Compressor

Проект реализует алгоритм **Huffman** для сжатия и восстановления текстовых и бинарных файлов на Java.

## Описание

Программа состоит из двух основных классов:

1. **HuffmanCompressor** – основной класс с методом `main`, который принимает аргументы командной строки:
   - `encode <входной файл> <выходной файл>` – сжимает файл
   - `decode <входной файл> <выходной файл>` – восстанавливает файл
2. **HuffmanProcessor** – класс, реализующий:
   - построение дерева Хаффмана  
   - генерацию таблицы кодов  
   - кодирование и декодирование данных  
   - работу с файлами  

Дополнительно есть вспомогательный класс **TreeElement**, описывающий узлы дерева Хаффмана.

## Сборка

1. Перейдите в папку проекта:


cd <путь_к_папке>/Huffman

Скомпилируйте проект:

javac HuffmanCompressor.java

Запуск возможен через командную строку:

java HuffmanCompressor <encode/decode> <входной файл> <выходной файл>

## Использование
Примеры:

Текстовый файл с 10 одинаковыми символами

echo "1111111111" > test1.txt
java HuffmanCompressor encode test1.txt output1.huff
java HuffmanCompressor decode output1.huff decoded1.txt


## Структура закодированного файла

Файл Хаффмана состоит из двух частей:

Закодированные данные – строка из битовых кодов (0 и 1), соответствующая исходному файлу.

Таблица кодов – строка TABLE, затем список соответствий символ → код:
1010101110
TABLE
49:0
50:10
51:11
